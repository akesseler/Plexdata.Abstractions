Remarks

This interface represents nothing else but the wrapper of HttpContext.

This interface has two significant differences compared to class HttpRequest. These differences are:

  1. The property Authentication is not included in this interface because it has been marked as deprecated in class HttpContext.
  2. The property User is of type IPrincipal instead of type ClaimsPrincipal. But if it is going to be set it must be of that type.
