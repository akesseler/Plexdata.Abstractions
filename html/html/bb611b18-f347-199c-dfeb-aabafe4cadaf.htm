Plexdata.AspNetCore.Http

In fact, testing of HTTP requests is difficult or even impossible in a .NET Core environment. Furthermore, at the moment it is unknown that a collection of HTTP context interfaces already exists. The Plexdata ASP.NET Core HTTP Abstraction library is an attempt to close this gap.

Framework

This library uses .NET Core 2.1.

Dependencies

This library depends on Microsoft.AspNetCore.Http 2.1.1.
